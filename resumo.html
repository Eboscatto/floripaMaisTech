Manipulação do DOM:

DOM - Doument Object Model.

Cada elemento de um documento HTML tem uma representação dentro do DOM.

Formas de selecionar elementos:

Por Id: document.getElementById("nomeId");

Por Class: document.getElementsByClassName("nomeDaClasse");

Por selector: document.querySelector(".nomeDaClasse");

Usa-se # para pegar pelo ID: document.querySelector("#nomeDoId");

OBS.: O querySelector pega qualquer elemento HTML.

Alterando propriedades:

Após selecionar um elemento podemos alterar suas propriedades:

  • Adidionar estilos:

    elemento.sytle.color="blue";

  • Adicionar/remover texto:

    elemento.innerText="olá, mundo!";

  • Alterar o HTML:

    elemento.innerHTML="

    Olá!

    >"

Adicionar e remover classes:

  • Adicionar classe:

    const div = document.getElementById('div-pai')

    div.classList.add("classe")

  • Remover classe:

    const div = document.getElementById('div-pai')

    div.classList.remove("classe")

Elementos com propriedades especiais como os inputs

const meuInput = document.getElementById("id-do-input")

console.log(meuInput.value); //mostra o valor digitado no input

meuInput.value = "Texto inserido no input!";
