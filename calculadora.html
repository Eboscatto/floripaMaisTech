Projeto Calculadora

  1. Executar o programa "Calculadora"
  2. Digitar o primeiro valor desejado
  3. Digiar o segundo valor desejado
  4. clicar no tipo de operação que deseja realizar. Ex.:multiplicação, divisão, etc.
  5. Clicar no botão "Processar" para obter o resultado
  6. Clicar no botão "Limpar" para limpar os campos
